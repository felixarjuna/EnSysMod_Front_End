# Getting started
## Installation

Open Terminal and type following command: 
```
git clone https://github.com/felixarjuna/EnSysMod_Front_End.git
cd EnSysMod_Front_End
```

Install all the package needed: 
```
npm i 
```

Run the web app
```
npm start
```
